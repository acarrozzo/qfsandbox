API Usage

Use UIComponents class for consistent button creation across the application.

// Primary Button
UIComponents.primaryButton('Save', { 
    icon: 'fas fa-save', 
    size: 'md' 
});

// Secondary Button  
UIComponents.secondaryButton('Cancel', { 
    size: 'sm' 
});

// Tertiary Button
UIComponents.tertiaryButton('New Project', { 
    icon: 'fas fa-plus', 
    size: 'lg' 
});

// With all options
UIComponents.primaryButton('Submit', {
    icon: 'fas fa-check',
    size: 'lg',
    disabled: false,
    className: 'custom-class',
    onClick: 'handleSubmit()'
});

Component Options

icon string (FontAwesome class)
size 'sm' | 'md' | 'lg'
disabled boolean
className string (additional CSS)
onClick string (function name)
